#Crochet Along

Crochet Along is my Static Webpage Assignment for Web Programming I.  I was tasked with creating a business website using HTML and CSS.
This assignment demonstrates my understanding of the following key concepts I learned in the course.

- HTML Basics
- Page Design
- Fonts & Colours
- Design
- CSS Selectors
- Box Model, Images & Graphics
- Responsive Web Design
- Flexbox & Grid
- CSS var/cal, Pseudo elements and classes
- Page Layout with CSS
- Tables
- Forms
- Multimedia
- CSS Libraries
- Publishing and Promoting

Using these techniques I was able to create this website.
